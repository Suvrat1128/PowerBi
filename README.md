📊 Project Overview


This project analyzes Spotify music streaming data to uncover insights about listener behavior, artist performance, and track characteristics. Using data-driven techniques, it identifies key factors that influence song popularity such as genre, audio features, and artist trends.

⚙️ Tech Stack

Python (Pandas, NumPy) → Data cleaning & analysis
Power BI → Interactive dashboards & visualization
EDA (Exploratory Data Analysis) → Pattern discovery & insights

🔄 Working

Data Collection – Spotify dataset with tracks, artists, and audio features
Data Preprocessing – Handling missing values, cleaning, and transformation
EDA – Analyzing genre trends, artist performance, and feature correlations
Visualization – Building interactive dashboards in Power BI
Insights Generation – Identifying patterns affecting music popularity


📈 Key Insights

Popularity is concentrated among a few top artists (long-tail distribution)
Genres like Pop show higher engagement
Audio features like energy positively impact popularity
Tempo and liveness have weaker influence


🎯 Objective


To build an analytics system that helps understand music trends and supports data-driven decision-making in the music industry.
